## HOW TO IMPLEMENT

**WordPress + Yoast SEO:**
- Install Yoast SEO plugin (free)
- On each page, scroll to the Yoast SEO block below the editor
- Enter the SEO title and meta description in the fields provided
- Preview shows how it'll look in Google results

**WordPress + Rank Math:**
- Same process — look for the Rank Math block below the editor

**Manually in theme (if no plugin):**
- Add inside `<head>` tags:
```html
<title>Page Title Here</title>
<meta name="description" content="Meta description here.">
```

**Rules reminders:**
- Title tags: max 60 characters (Google truncates at ~580px)
- Meta descriptions: 150–160 characters (longer gets truncated)
- No duplicate titles or descriptions across any two pages
- Keywords should appear naturally — don't stuff
